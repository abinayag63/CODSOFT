

📌 Project Overview

This repository contains four beginner-friendly Machine Learning projects developed using Python. Each project demonstrates the complete machine learning workflow, including data preprocessing, model training, prediction, evaluation, and data visualization. These projects provide hands-on experience with both classification and regression techniques using real-world datasets.

🎯 Objectives

* Understand the fundamentals of Machine Learning.
* Perform data preprocessing and feature preparation.
* Build classification and regression models.
* Evaluate model performance using appropriate metrics.
* Visualize data and prediction results.

 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

📂 Projects 

 🚢 Task 1: Titanic Survival Prediction

Predicts whether a passenger survived the Titanic disaster using a Random Forest Classification model.

 🎬 Task 2: Movie Rating Prediction

Predicts IMDb movie ratings using a Random Forest Regression model based on movie-related attributes.

 🌸 Task 3: Iris Flower Classification

Classifies Iris flowers into Setosa, Versicolor, and Virginica species using a Machine Learning classification model.

 📈 Task 4: Sales Prediction

Predicts product sales based on advertising expenditure using a Linear Regression model.

 📚 What I Learned

* Data preprocessing and cleaning.
* Handling missing values and encoding categorical data.
* Training classification and regression models.
* Model evaluation using Accuracy, MAE, MSE, and R² Score.
* Data visualization using charts and graphs.
* Building complete machine learning workflows using Python.

 ✅ Conclusion

These projects strengthened my understanding of machine learning concepts and provided practical experience in solving real-world prediction and classification problems using Python and Scikit-learn.
